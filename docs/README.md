### Installation

#### ✨ Latest release

Create an account at [Uindow](https://uindow.com/?ref=uindow.github.io) and follow the on-screen instructions.

#### 📦 Older release

To install an older version, visit the [Releases](https://github.com/uindow/uindow/releases/) page and download the appropriate installer.

#### 🌙 Nightly build

To run the latest (unreleased) version of Uindow, clone [this repository](https://github.com/uindow/uindow/) and run the following commands:

```bash
npm install
npm start
```
